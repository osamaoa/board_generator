# Documentation

Use these documents for the public package:

- `docs/paper.pdf`: theory reference for the released generator. The PDF will be added before release.
- `docs/web_app.md`: UI manual for interactive generation, visualization, and exports.
- `docs/cli.md`: CLI manual for batch exports, knot-model workflows, and photorealistic diffusion training.

Additional setup notes are in `docs/getting_started.md`.

The JSON files in this folder are batch-generation examples for `boards generate`.
